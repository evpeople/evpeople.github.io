01分数规划

一堆物品，有一个收益，有一个代价，给出一个方案，使得收益/代价 最大。

  1. 首先证明以单个的受益/代价 的贪心是错的
  2. 先假设最大值大于等于x
  3. 然后移项可得
    1. 收益-x代价大于等于0
    2. 所以贪心应为收益-x代价必须全选正的，
    3. 所以二分x，看求和结果是不是大于x
    4. 若不成立，说明这个x取大了
    5. x为和，为最后的结果
  4. 若为不是若干个而是k个，则应该让大于零的数应该有k个
