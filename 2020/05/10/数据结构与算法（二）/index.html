第二天学习的内容，关于欧几里得算法求最大公因数，以及快速幂

欧几里得算法

int gcd(int m ,int n)
{
    int rem;
    while (n>0)
    {
        rem=m%n;
        m=n;
        n=rem;
    }
    return m;
}

称之为辗转相除法也可。时间复杂度为logn

通过定理

如果M>N,则M mod N < M/2

分两种情况证明

快速幂

int qiumi(long int x,int n)
{
    //复杂度为log（n）
    if (n==0)
    {
        return 1;
    }
    if (n==1)
    {
        return x;
    }
    if (n%2==0)//偶数
    {
        return qiumi(x*x,n/2);//求x方的n/2次幂。然后x的四次方的n/4次幂，以此类推，得到x的n次幂。
    }else
    {
        return qiumi(x*x,(n)/2)*x;
    }
}

时间复杂度为 logn

一般的循环做法时间复杂度为n

求2^10次方，可以变为2^5 *2 ^5

然后2^5 可以为2 ^2 *2^2 *2

也就是先计算2×2，然后 4×4×2，然后32×32，总共四次乘法

然后写递推的思路，求x^n，若n为偶数，则先求 x^(n/2) ，然后对其 平方。

然后如果n为奇数，则求x的n-1次方，再乘x
