题干

一个正整数序列共有N个数，现将它们依次划分成M份。划分的要求是序列原有的顺序不变，每份中的所有的数位置要连续。请你写一段程序计算一下这M份中和最大的那一份最小是多少？

样例解释：这5个划分分别为，（10，40）、（30，10）、（50）、（11）、（40）。这种情况下最大的和为50，其他方式最大的那一份的和均大于50。

输入：
第一行为用空格分隔的2个整数，分别代表N和M。（1 <= N <= 100000 ，1 <= M <= N），接下来N行，每行为一个整数，（这些整数均大于等于1且小于等于10000）。

输出：
只有一行，为一个整数，代表此条件下所求得的最小值。

提示：二分查找

来源buptoj

想法

怎么二分

  已经给了二分查找的提示，那么首先应该确定二分的范围，二分的对象，二分判断的内容。由题意不难看出用排除法二分对象因该是和。那么就是去找和的最大值和最小值。最大的话当然就是所有整数的和了，最小应该是正整数序列中最大的那个。二分应该判断什么呢？反复分析使用排除法后找出。emmmm不是那么直观，先往后写嘛
  所以现在我们的想法就是先求出一个和 先不管怎么求出 然后二分判断。但是还是不知道判断的内容是什么，所以先考虑别的？
  考虑求和函数，设计函数头，返回值。

  1. 传入数组
  2. 目标和，也就是加到多少停止。
  3. 返回份数
      这时候二分的判断条件也就浮出水面，份数，当我在某个和下需要分大于所要求的份数才能分出某个和，说明这个和小了。
     > 100个1，40为某个和 2为所需份数，分3份才能出结果，比2大，说明40小了

    怎么求和

      for循环从头加，在内部进行判断和是否大于了所要求，大于则说明分出了一组，然后和归零重来

AC代码

#include<stdio.h>

int n,m;
int a[100000];
int check(int M){
    int group=0,now=0;
    for(int i=0;i<n;i++){
        if(a[i]>M) return 0;
        if(now+a[i]>M){
            group++;
            now=0;    
        }    
        now+=a[i];
    }
    return group<m;
}
int main(){
    scanf("%d%d",&n,&m)
    int R=0,L=0;
    for(int i=0;i<n;i++){
        scanf("%d",%a[i])
        R+=a[i];
        L=L>a[i]?L:a[i];
    }
    R++;    
    while(L<R){
        int M=(L+R)/2;
        if(check(M)) R=M;
        else L=M+1;
    }
    printf("%d",L);
    return 0;
}
